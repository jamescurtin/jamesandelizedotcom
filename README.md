## Developing

-   To start up app:

    ```bash
    docker-compose up app
    ```

    then navigate to `localhost:8000`.

-   To run `yarn` commands:

    ```bash
    docker-compose run --rm yarn <COMMAND>
    ```

-   To drop into a shell:

    ```bash
    docker-compose run --rm shell
    ```
